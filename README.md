# 🚖 NYC Taxi Trip Analysis

## 📌 Project Overview

This project focuses on Exploratory Data Analysis (EDA) of NYC Taxi trip data to identify trip patterns, fare behavior, demand trends, and operational inefficiencies.

---

## 📂 Project Files

### requirements.txt

Contains all Python libraries and dependencies used in the project.

### dataset.csv

Original raw NYC taxi dataset used for analysis.

### NYC_Taxi_Trip_Analysis.ipynb

Complete Jupyter Notebook containing:

* Data Cleaning
* Feature Engineering
* EDA
* Visualizations
* Business Insights

### cleaned_dataset.csv

Cleaned dataset after preprocessing and handling missing values/outliers.

### images/

Contains important visualization screenshots generated during analysis.
